# Mise à jour v251208

## Ajout d'un champ de recherche sur la sélection d'utilisateur

Dans l'ensemble de CertiSoft, lors de la sélection d'un utilisateur, vous avez la possibilité de filtrer la liste en fonction d'un critère de recherche.  
![filter_user.png](filter_user.png)

## Duplication d'audit

Vous avez désormais la possibilité de dupliquer un audit complet. Cliquez sur le bouton indiqué dans la capture d'écran ci-après.  
![duplicate_audit.png](duplicate_audit.png)

## Terminer une action sans tâches

Une action peut être terminée même si elle ne contient pas de tâches, ou que toutes les tâches ne sont pas terminées.  
![terminate_action.png](terminate_action.png)

## Impression de la cartographie

La cartographie peut désormais être imprimée avec la date et l'heure de l'impression, ainsi que le nom de l'utilisateur ayant lancé l'impression.  
![cartoghraphie_print.png](cartoghraphie_print.png)

## Tâches visibles pour les utilisateurs liés

Les utilisateurs faisant partie d'une tâche, mais pas de l'action, ont désormais la possibilité de visualiser cette dernière dans l'application web.

## Bugs corrigés

- Récupération de mot de passe.
- Masquage du changement de mot de passe pour les utilisateurs de type SSO.
